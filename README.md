# Basketball Holístico

Una aplicación integral para la mejora del juego de baloncesto, enfocada en el desarrollo holístico del jugador.

## Características Principales

- Perfil del Jugador y Personalización
- Módulos de Entrenamiento Físico
  - Entrenamiento Pliométrico
  - Entrenamiento de Fuerza
  - Resistencia y Acondicionamiento
  - Flexibilidad y Movilidad
- Módulos de Habilidades Técnico-Tácticas
  - Dribling
  - Tiro
  - Pases y Recepciones
  - Posicionamiento Defensivo
- Preparación Mental
- Prevención de Lesiones
- Nutrición Deportiva
- Psicología del Equipo y Liderazgo

## Tecnologías Utilizadas

- React
- TypeScript
- Material-UI
- Redux Toolkit
- React Router
- Framer Motion

## Instalación

1. Clonar el repositorio
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Estructura del Proyecto

```
src/
  ├── components/     # Componentes reutilizables
  ├── features/       # Módulos principales de la aplicación
  ├── layouts/        # Layouts y plantillas
  ├── pages/         # Páginas principales
  ├── services/      # Servicios y APIs
  ├── store/         # Estado global (Redux)
  ├── styles/        # Estilos globales
  └── utils/         # Utilidades y helpers
```
